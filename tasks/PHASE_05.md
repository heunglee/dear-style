# Phase 05 — Review Loop

## Goal

Support self and friend A/B review.

## Tasks

- Create application_results table and API.
- Create review_sessions table and API.

## Acceptance Criteria

- User can upload before/after result.
- User can create shareable review link.
- Friend can vote without account.
- Report summarizes votes without attractiveness scoring.
