# Phase 03 — Trace Practice Engine

## Goal

Implement finger/stylus tracing practice.

## Tasks

- Implement backend trace scoring service.
- Implement trace attempt API.
- Implement web TraceCanvas.
- Implement Android trace view.
- Implement iOS trace view.
- Store practice sessions and attempts.
- Show score and feedback after each attempt.

## Acceptance Criteria

- User can trace an eyeliner target line.
- Scores include angle, length, endpoint, smoothness, and total.
- Feedback is understandable and non-judgmental.
