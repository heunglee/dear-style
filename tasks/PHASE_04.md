# Phase 04 — Mirror-Assisted Application

## Goal

Help users apply practiced techniques in real life.

## Tasks

- Add mirror mode screen using front camera.
- Add step checklist overlay.
- Add non-precision guidance text.
- Add before photo capture.
- Add after photo capture.

## Acceptance Criteria

- User can open mirror mode after practice.
- User can follow steps while using the front camera as mirror.
- User can capture before and after images.
