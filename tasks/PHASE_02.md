# Phase 02 — Eyeliner Lesson Catalog

## Goal

Implement reusable eyeliner learning content.

## Tasks

- Create style_lessons table.
- Create drawing_templates table.
- Seed initial eyeliner lessons.
- Build lesson list API.
- Build lesson detail API.
- Build web lesson list and detail screens.
- Build mobile lesson list and detail screens.

## Acceptance Criteria

- Users can browse lessons by impression and difficulty.
- Each lesson has steps, target impression, common mistakes, and practice CTA.
