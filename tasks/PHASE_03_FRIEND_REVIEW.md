# Phase 03 — Friend Review Tasks

## Goal

Add shareable friend review flow.

## Backend Tasks

- Implement friend_review_links table.
- Generate secure review tokens.
- Hash tokens in database.
- Implement public friend review retrieval endpoint.
- Implement friend review submission endpoint.
- Add expiry and max-review checks.
- Add rate limiting.

## Web Tasks

- Build share screen.
- Build public friend review page.
- Add report section for friend results.

## Android Tasks

- Add native share sheet integration.
- Show friend review status.

## iOS Tasks

- Add native share sheet integration.
- Show friend review status.

## Acceptance Criteria

- User can create friend review link.
- Friend can vote without login.
- User report shows self vs friend preference.
- Expired links cannot be used.
