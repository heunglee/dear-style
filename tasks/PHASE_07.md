# Phase 07 — Face and Eye Shape Personalization

## Goal

Introduce optional personalization without overclaiming.

## Tasks

- Add user self-assessment for eye shape and face shape.
- Add optional landmark-based suggestions later.
- Add recommendation rules by shape.
- Allow user correction.

## Acceptance Criteria

- Recommendations can mention why a technique may suit a given eye/face shape.
- User can override detected or suggested categories.
