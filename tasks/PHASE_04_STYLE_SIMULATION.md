# Phase 04 — Style Simulation Tasks

## Goal

Expand from color into makeup shape and visual impression.

## Backend Tasks

- Add session types for eyeliner_shape, lip_shape, and brow_shape.
- Add style element metadata.
- Extend report schema for impressions.

## Web Tasks

- Add test cards for eyeliner, lip line, and brow shape.
- Implement basic overlay prototypes.
- Add impression descriptors.

## Android Tasks

- Add style test selection.
- Render static style comparisons.

## iOS Tasks

- Add style test selection.
- Render static style comparisons.

## Acceptance Criteria

- User can compare at least two eyeliner shapes.
- Report explains how selected shape changes impression.

## Notes

Do not attempt photorealistic virtual try-on in this phase unless the MVP comparison engine is stable.
