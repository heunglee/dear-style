# Phase 03 — Makeup Coach MVP

## Goal

Generate actionable lip line, eyebrow, and blush guidance from facial geometry.

## Tasks

- Add coaching_sessions table and API.
- Add coaching_guides table and API output.
- Implement lip line guide generation.
- Implement eyebrow start, arch, and tail guidance.
- Implement blush placement and blending guidance.
- Add overlay primitive schema.
- Add confidence and safety warnings to all guide outputs.

## Acceptance Criteria

- A user can request a guide for lip line, eyebrow, or blush.
- Guide output includes steps, overlay primitives, and confidence.
- Guidance remains non-judgmental and action-oriented.
