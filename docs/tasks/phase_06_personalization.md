# Phase 06 — Personalization

## Goal

Improve coaching with user preferences, correction history, and optional self-assessment.

## Tasks

- Add editable user style goals.
- Add optional face/feature self-assessment.
- Add history-aware coaching suggestions.
- Let users correct detected categories.
- Track repeated feedback themes.

## Acceptance Criteria

- User can override or correct inferred attributes.
- Coaching can adapt to saved preferences.
- System clearly distinguishes user-provided data from detected data.
