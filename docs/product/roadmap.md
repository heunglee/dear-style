# Roadmap

## Phase 1 — Foundation

Goal: build the platform foundation and align the repository around the v2 coaching pivot.

Features:

- accounts
- consent and privacy foundation
- image upload metadata
- coaching session model
- agent contract documentation
- cross-platform app foundations

## Phase 2 — Face Analysis MVP

Goal: extract enough facial geometry to support coaching.

Features:

- image quality validation
- face detection
- landmark extraction
- face shape and feature localization
- geometry confidence scoring

## Phase 3 — Makeup Coach MVP

Goal: generate actionable guides from facial geometry.

Features:

- lip line guide
- eyebrow start, arch, and tail guide
- blush placement map
- overlay generation
- correction feedback copy

## Phase 4 — Feedback and Release

Goal: close the before/after learning loop and ship a cohesive first release.

Features:

- before/after comparison
- self review
- AI-assisted feedback summary
- quality and confidence warnings
- safety and privacy controls
- cross-platform release checklist

## Phase 5 — Recommendation Expansion

Goal: add style and occasion recommendations once coaching is useful.

Features:

- occasion guidance
- style strategy recommendations
- product category recommendations
- user preference memory

## Phase 6 — Practice and Progress

Goal: help users improve over time.

Features:

- progress history
- repeated feedback comparison
- saved looks
- guided routines

## Phase 7 — Appearance Graph

Goal: model relationships among features, makeup techniques, recommendations, feedback, and outcomes.

Features:

- graph schema
- preference graph
- coaching graph
- impression graph

## Phase 8 — Color Exploration

Goal: add cautious color education.

Features:

- color preference quizzes
- controlled photo capture guidance
- low-confidence color harmony experiments
- no definitive undertone diagnosis

## Phase 9 — Product Matching

Goal: connect coaching strategies to product categories.

Features:

- product category taxonomy
- category-level recommendation API
- coaching target to product category mapping
- Dear Daughter Beauty product matching only when explicitly approved

## Phase 10 — Brand Insights

Goal: aggregate anonymized learning and preference data for product development.

Features:

- coaching target demand clusters
- common feedback themes
- product gap discovery
- style and occasion trend reporting
