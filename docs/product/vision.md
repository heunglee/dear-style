# Dear Style Vision

Most beauty applications focus on:

```text
What color should I buy?
```

Dear Style focuses on:

```text
How should I apply makeup?
```

The application acts as a digital makeup coach.

Users upload a selfie.

The platform:

- identifies facial landmarks
- analyzes facial geometry
- generates coaching overlays
- provides improvement feedback

The goal is skill development.

Not product selling.

## Current Pivot

The v2 direction is face-geometry-based coaching. The core product should help users understand placement, shape, direction, and correction for their own facial features.

Initial coaching domains:

- lip line
- eyebrows
- blush placement
- before/after feedback

Color matching, product recommendations, and virtual try-on can exist later, but they should not define the MVP.
