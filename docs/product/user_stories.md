# User Stories

## Beginner Makeup User

- As a beginner, I want to upload a selfie and receive simple placement guidance so I know where to start.
- As a beginner, I want the app to explain why a suggested lip line or blush area fits my face so I can learn the principle.
- As a beginner, I want feedback to be gentle and specific so I can improve without feeling judged.

## Daily Makeup User

- As a daily makeup user, I want repeatable eyebrow and blush guidance so my application is consistent.
- As a daily makeup user, I want quick correction feedback from before/after photos so I can adjust faster.
- As a daily makeup user, I want saved preferences so the app remembers my usual style goals.

## Beauty Enthusiast

- As a beauty enthusiast, I want to explore different style strategies so I can experiment intentionally.
- As a beauty enthusiast, I want occasion-based recommendations so I can adapt my makeup for context.
- As a beauty enthusiast, I want more advanced feedback when image confidence is high.
