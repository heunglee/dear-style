# User Personas

## Persona 1

Beginner Makeup User

Pain Points:
- Doesn't know where to start.
- Watches tutorials but cannot replicate results.
- Struggles with symmetry.

Goals:
- Simple guidance.
- Step-by-step instructions.

## Persona 2

Daily Makeup User

Pain Points:
- Wants consistency.
- Wants faster application.

Goals:
- Personalized guides.
- Efficiency.

## Persona 3

Beauty Enthusiast

Pain Points:
- Wants experimentation.

Goals:
- Style exploration.
- Advanced techniques.
