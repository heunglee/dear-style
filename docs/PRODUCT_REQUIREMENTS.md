# Product Requirements

## Product Name

Dear Style

## Parent Brand

Dear Daughter Beauty

## Product Category

Makeup drawing coach, appearance design learning platform, and style practice system.

## MVP Goal

Help users learn and practice eyeliner and lip-line techniques that create desired visual impressions.

## Non-Goals for MVP

- Definitive undertone diagnosis
- High-fidelity virtual try-on
- Gold/silver diagnostic scoring
- Clothing or eyewear AR compositing
- Product sales optimization
- Attractiveness scoring

## Core User Flow

1. User selects an impression goal.
2. System recommends a makeup line style.
3. User learns the geometry and sequence.
4. User practices the line by tracing on the screen.
5. User applies the look using a mirror or front camera reference.
6. User uploads before/after photos.
7. User can ask friends to review A/B comparisons.
8. System summarizes learning progress and next practice steps.

## Impression Goals

Supported initial goals:

- Natural
- Professional
- Friendly
- Soft
- Lifted
- Sharp
- Elegant
- Bold

## Eyeliner Styles

Initial style catalog:

- Natural Tightline
- Soft Outer Third Line
- Soft Cat Eye
- Classic Cat Eye
- Puppy Eye
- Horizontal Extension
- Thin Lifted Line

Each style must include:

- intended impression
- best-fit contexts
- drawing steps
- practice template
- common mistakes
- safety notes

## Lip-Line Styles

Optional MVP+ catalog:

- Natural Lip Line
- Soft Blurred Lip
- Center Overline
- Defined Cupid Bow
- Balanced Edge Lip

## Trace Practice Requirements

The app must let users:

- see a target line
- trace the line with finger or stylus
- repeat practice attempts
- receive feedback on angle, length, smoothness, and endpoint accuracy
- view progress across attempts

## Friend Review Requirements

Friend review compares one user's A/B looks.

Allowed question examples:

- Which look feels more professional?
- Which look feels softer?
- Which look feels more lifted?
- Which look feels more natural?

Disallowed question examples:

- Which person is prettier?
- Rate attractiveness.
- Rank friends.

## Color Feature Position

Color harmony and undertone concepts may be included as educational or exploratory features only. The product should communicate that digital color analysis is limited without controlled lighting and real materials.
