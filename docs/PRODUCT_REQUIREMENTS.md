# Product Requirements

## Scope

Dear Style supports consumer-facing color and style discovery, with future B2B data insights for beauty product development.

## Core MVP Features

### 1. User Account

Users can create an account, sign in, manage consent, and view previous reports.

### 2. Consent and Privacy Flow

Before uploading face images, users must see a clear consent screen covering:

- what is uploaded
- why it is used
- whether raw images are stored
- how to delete data
- how friend review links work

### 3. Face Image Upload

Users upload or capture a face photo.

The system should support metadata:

- lighting condition, if self-reported
- indoor/outdoor
- camera source
- timestamp
- optional notes

### 4. A/B Comparison Session

The system generates comparison cards using the same user image with different overlays.

Initial comparison types:

- Gold vs Silver
- Ivory vs Pure White
- Camel vs Cool Grey
- Gold Frame vs Silver Frame
- Tortoise Frame vs Black Frame
- Coral Lip vs Rose Lip
- Brick Lip vs Berry Lip

### 5. Self Review

Users choose which option appears:

- healthier
- more natural
- brighter
- more harmonious
- more polished

Do not ask which option makes the user prettier.

### 6. Friend Review

Users can share a review link.

Friends can vote without creating an account.

Friend review should focus on the same appearance descriptors:

- healthier
- more natural
- brighter
- more harmonious
- more polished

### 7. Result Report

The report should include:

- self preference summary
- friend preference summary
- comparison consistency
- warm/cool/neutral tendency
- brightness tendency
- saturation tendency
- contrast tendency
- recommended next tests

### 8. Education

MVP includes lightweight explanations for:

- undertone
- color harmony
- warm vs cool
- brightness
- saturation
- contrast

## Secondary Features

### Makeup Shape Discovery

Future comparison types:

- natural eyeliner
- cat eye
- puppy eye
- fox eye
- straight brow
- soft arch brow
- blurred lip line
- sharp lip line
- over lip

### Product Matching

Future product recommendations:

- lipstick
- foundation
- eyeliner
- blush
- eyewear
- clothing colors

### Brand Insight

Future anonymized aggregate analytics:

- shade demand
- preference clusters
- product gaps
- trend signals

## Non-Goals for MVP

- full AI makeup try-on realism
- medical skin diagnosis
- attractiveness scoring
- race or ethnicity inference
- automated shopping checkout
- B2B dashboard
