# UI/UX Spec

## Product UX Thesis

Dear Style should feel like a patient beauty coach, not a beauty judge.

Core flow:

```text
Capture → Analyze → Guide → Apply → Review → Improve
```

## Main Navigation

- Home
- Coach
- Feedback
- Profile

## Home Screen

Purpose:

- introduce current goal
- continue latest coaching session
- show recommended coaching targets

Primary CTA:

- Start Makeup Coach

## Coaching Target Screen

User selects a target area:

- Lip line
- Eyebrow
- Blush
- Before/after feedback

## Selfie Capture Screen

Sections:

- photo guidance
- privacy reminder
- upload or capture
- image quality result

## Coaching Result Screen

Requirements:

- visible overlay or placement map
- step-by-step instructions
- confidence notes
- save or retake action
- apply and compare action

## Mirror Mode Screen

Purpose:

- let front camera act as a mirror
- show textual checkpoints
- optionally show reference dots or a non-precision guide

Warning text:

"Use this as a guide, not a fixed overlay. Keep your face steady and follow the step sequence."

## Feedback Screen

Purpose:

- compare before/after images
- summarize placement observations
- provide next-step corrections
- show confidence and image quality warnings

## Application Upload Screen

Allows:

- before photo
- after photo
- notes
- request feedback

## Report Screen

Shows:

- before/after summary
- placement observations
- improvement suggestions
- confidence warnings
- next coaching recommendation

## Color Exploration UX

Deferred. Must show limitation notice if enabled.
