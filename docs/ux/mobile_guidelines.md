# Mobile Spec

## Shared Mobile Requirements

- Native camera access
- Image upload
- Coaching overlay rendering
- Before/after capture
- Haptic feedback only when it clarifies a user action
- Offline draft support for pending coaching or feedback submissions

## Mirror Mode

Mirror mode uses the front camera. It should not promise precise AR alignment in the MVP.

Features:

- front camera preview
- step checklist
- optional static reference line
- capture before/after

## Coaching Mode

Coaching mode should make guidance easy to follow while the user is holding a phone.

Requirements:

- large readable instructions
- simple overlay controls
- clear confidence or image quality messages
- one primary action per screen when possible
- safe camera/photo permission copy
