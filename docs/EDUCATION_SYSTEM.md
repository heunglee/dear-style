# Education System

## Purpose

Help users understand beauty and style terminology through visual explanations.

## MVP Education Topics

- undertone
- color harmony
- warm vs cool
- brightness
- saturation
- contrast

## Key Messaging

Undertone does not mean the visible surface color of the skin.

It is better explained as a tendency in which some color families appear more harmonious than others.

## Example Explanation

Warm tone does not simply mean yellow skin.

A warm-leaning result means warm colors such as gold, ivory, camel, peach, or coral may appear more harmonious in repeated comparisons.

## Future Topics

- eyeliner shape
- lip line effects
- eyebrow shape
- eyewear frame color
- event styling

## Content Format

Each education unit should have:

- title
- short explanation
- visual example
- common misconception
- test suggestion

## Safety

Do not frame education as rules about what the user can or cannot wear.

Frame it as tools for intentional styling.
