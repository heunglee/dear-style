# Dear Style

AI Makeup Coaching Platform

## Mission

Help users apply makeup better, not simply choose colors.

## Primary Value

Guide users through lip line, eyebrow shaping, blush placement, and application feedback using AI-assisted facial analysis.

## Core Principle

Actionable guidance beats static analysis.

The product should prioritize:

1. Coaching
2. Visualization
3. Step-by-step guidance
4. Correction feedback

over:

1. Beauty scoring
2. Generic recommendations
3. Personality-style quizzes
4. Product-first selling

Every feature must answer:

```text
What action can the user take next?
```
