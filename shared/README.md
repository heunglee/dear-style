# Shared

Shared contracts and cross-platform conventions for Dear Style.

## Current MVP Contracts

- `api/type_conventions.md`: human-readable API type conventions
- `api/type_catalog.json`: machine-readable enum and DTO catalog

These files define the naming and value conventions that backend, web, Android, and iOS should
mirror until generated clients are introduced.

## Deferred References

- `TRACE_SCORING_SPEC.md` is retained as a deferred trace-practice reference and is not part of the
  coaching MVP runtime API.
