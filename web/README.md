# Dear Style Web

Next.js web app for the Dear Style MVP.

## Local Setup

```bash
cd web
npm install
cp .env.example .env.local
npm run dev
```

The app expects the backend at `NEXT_PUBLIC_API_BASE_URL`.

## Implemented Phase 01 Surface

- `/` welcome flow
- `/signup`
- `/login`
- `/consent`
- API client for auth and consents
